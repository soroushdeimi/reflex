# پروژه Reflex - پیاده‌سازی کامل

## شماره دانشجویی
400108582, 
400108955,
400108966

## توضیحات
این پروژه پیاده‌سازی کامل پروتکل Reflex برای Xray-Core است. پروتکل Reflex با هدف غیرقابل تشخیص کردن ترافیک پراکسی طراحی شده است.

### قابلیت‌های پیاده‌سازی شده:
- **Step 1**: ساختار کامل پکیج و تنظیمات در Xray-Core.
- **Step 2**: هندشیک ضمنی با استفاده از X25519 و احراز هویت با UUID.
- **Step 3**: رمزنگاری فریم‌ها با ChaCha20-Poly1305 AEAD.
- **Step 4**: مکانیزم Fallback کامل برای هدایت ترافیک غیر Reflex به وب‌سرور (مشابه Trojan).
- **Step 5**: قابلیت Traffic Morphing پیشرفته با پروفایل‌های YouTube، Zoom و HTTP/2 API. پشتیبانی از فریم‌های کنترلی PADDING و TIMING.

## نحوه اجرا
1. پیش‌نیازها: Go 1.18 یا بالاتر.
2. بیلد پروژه:
   ```bash
   cd xray-core
   go build -o xray ./main
   ```
3. اجرای سرور با فایل کانفیگ مثال:
   ```bash
   ./xray -config ../config.example.json
   ```

## تست‌ها
تمامی تست‌های واحد و یکپارچگی با موفقیت پاس می‌شوند:
```bash
cd xray-core
go test ./proxy/reflex/... ./tests/... -v
```

## مشکلات و راه‌حل‌ها
- **مشکل در تولید کد Protobuf**: به دلیل عدم دسترسی به `protoc` در برخی محیط‌ها، اینترفیس `proto.Message` به صورت دستی برای ساختارهای تنظیمات پیاده‌سازی شد تا با سیستم داخلی Xray سازگار باشد.
- **یکپارچه‌سازی با هسته Xray**: برای پشتیبانی از تنظیمات JSON، مبدل‌های لازم در پکیج `infra/conf` اضافه شدند.
